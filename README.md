# Production Management Data Analysis Project

## 프로젝트 개요
1. 생산관리 교육 과정에서 수행한 데이터 분석 과제들을 정리한 프로젝트입니다.  
2. Python을 활용하여 생산관리 데이터를 분석하고 공정 효율성, 품질, 생산계획, 재고관리, 원가관리 등 제조 운영 의사결정을 분석한 프로젝트입니다.
3. 각 실습에서는 실제 생산공정 데이터를 기반으로 공정 분석 및 운영 의사결정 문제를 해결하는 것을 목표로 하였습니다.

## 수행 과제
1. Line Balancing Analysis : 생산라인의 공정별 작업시간 데이터를 분석하여 병목공정을 도출하고 공정 효율성을 개선하는 분석을 수행했습니다.
Analysis Process
- 공정별 평균 작업시간 계산
- 제품 생산에 필요한 총 공정시간 계산
- 공정별 작업시간 비교를 통한 병목공정 도출
- 공정 효율성(Line Efficiency) 계산
- 병목공정의 생산능력 개선 시 공정 효율 변화 분석
Key Result
- C 공정이 병목공정으로 식별됨
- 병목공정 개선 시 공정 효율이 크게 향상됨
- 공정시간 균형이 생산 효율성에 미치는 영향을 확인

2. Defect Rate Analysis : 생산 공정에서 발생하는 불량 데이터를 분석하여 공정별 불량률과 실제 공정 소요시간을 계산했습니다.
Analysis Process
- 공정별 평균 공정시간 계산
- 투입량 대비 산출량을 기반으로 불량률 계산
- 불량률을 반영한 기대 공정시간 계산
- 공정 효율성 분석
Key Result
- 불량률이 공정 기대 소요시간에 직접적인 영향을 미침
- 불량률을 고려한 공정 분석이 실제 생산라인 효율 평가에 중요함

3. Production Scheduling : BOM 데이터를 기반으로 생산에 필요한 작업시간을 계산하고 인력 운영 계획을 분석했습니다.
Analysis Process
- 공정별 평균 작업시간 계산
- BOM 기반 표준공수 산출 
- 작업지시 수량을 반영한 총 작업시간 계산
- 가용 인력과 작업시간 비교
- 필요 인력 산정
Key Result
- 총 작업시간이 가용 인력시간보다 커 추가 인력이 필요함
- 생산 스케줄링 시 작업시간과 인력 계획이 중요함

4. Inventory Management (EOQ) : 경제적 주문량(EOQ) 모델을 적용하여 재고 운영 전략을 분석했습니다.
Analysis Process
- EOQ 모델을 이용한 경제적 주문량 계산
- 공정 데이터 기반 병목공정 식별
- 생산능력 기반 재주문점(Reorder Point) 계산
- 수요 기반 재주문점 계산
- 최종 재주문점 도출
Key Result
- EOQ = 200
- 생산능력을 고려한 재주문점이 재고 운영 의사결정에 중요한 요소로 확인됨

5. Manufacturing Cost Analysis : BOM 구조와 공정 데이터를 활용하여 제조원가를 계산하고 외주 생산 여부를 분석했습니다.
Analysis Process
- 공정별 평균 작업시간 기반 표준공수 산출
- 직접 노무비 계산
- 간접 노무비 계산
- 재료비 계산
- 제조원가 계산
- 외주 비용과 비교하여 생산 방식 결정
Key Result
- 공정 누적 제조원가 계산
- 외주 비용과 비교 결과 직접 생산이 경제적
