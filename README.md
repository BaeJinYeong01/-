# Production Management Data Analysis Project

## 프로젝트 개요
1. 생산관리 교육 과정에서 수행한 데이터 분석 과제들을 정리한 프로젝트입니다.  
2. Python을 활용하여 생산관리 데이터를 분석하고 공정 효율성, 품질, 생산계획, 재고관리, 원가관리 등 제조 운영 의사결정을 분석한 프로젝트입니다.
3. 각 실습에서는 실제 생산공정 데이터를 기반으로 공정 분석 및 운영 의사결정 문제를 해결하는 것을 목표로 하였습니다.

## 수행 과제
1. EOQ(Economic Order Quantity) 모델을 활용한 경제적 주문량 분석
2. 생산 라인 밸런싱(Line Balancing) 계산 및 공정 효율 분석
3. 재주문점(ROP, Reorder Point) 계산 모델 구현
4. 수요 데이터를 활용한 재고 관리 분석
5. 생산 데이터 기반 공정 효율성 분석

## 분석 내용
- EOQ 모델을 활용한 최적 주문량 계산
- 생산 라인 작업 시간 기반 라인 밸런싱 분석
- 재고 수요와 리드타임을 고려한 재주문점 계산
- 생산 데이터를 기반으로 공정 효율 분석
- 데이터 기반 생산 및 재고 관리 의사결정 모델 구현

## 사용 기술
- Python
- Pandas
- Numpy
- Excel

## 사용 데이터
생산관리 교육 과정에서 제공된 생산 데이터셋을 활용하여 분석을 수행하였습니다.

## 프로젝트 의의
본 프로젝트는 생산관리 이론을 실제 데이터 분석과 연계하여  
생산 효율 개선과 재고 관리 의사결정을 지원하는 분석 모델을 구현하는 것을 목표로 수행되었습니다.  
이를 통해 생산 데이터 기반 의사결정의 중요성과 생산관리 분석 기법을 실습할 수 있었습니다.
